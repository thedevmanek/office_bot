# Asset Attribution And License Review

This repository includes Gazebo/Ignition world assets under
`dev_ws/src/office_bot_model/models/worlds/office_world/`. The source
`model.config` files record authorship metadata, but the checked-in
`model.config` files do not declare explicit license fields. Treat these assets
as attribution-recorded and license-review-required before commercial
redistribution, hardware product packaging, or reuse outside this repository.

The OpenHRI `office_bot` project source code is Apache-2.0 under
[LICENSE](../LICENSE). That project license does not change or infer terms for
third-party mesh, texture, animation, or world assets whose local model metadata
does not include a license declaration.

## Shared World Media

The office world also includes shared material, texture, OBJ, and MTL assets in
two mirrored media trees:

- `dev_ws/src/office_bot_model/models/worlds/office_world/media/`
- `dev_ws/src/office_bot_model/models/worlds/office_world/models/media/`

Those directories do not contain local `model.config` metadata, author fields,
or license fields. Treat them as part of the same license-review-required
simulation asset bundle.
